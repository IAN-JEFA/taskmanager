# Task-Management# Task Manager Pro - Complete Task Management System

## Features Implemented

### Core Features
1. ✅ **Create Tasks** - With validation for duplicate titles on same date, future dates only
2. ✅ **List Tasks** - Sort by priority (high → low) then due date ascending
3. ✅ **Update Task Status** - Progressive workflow: pending → in_progress → done
4. ✅ **Delete Tasks** - Only completed (done) tasks can be deleted
5. ✅ **Daily Task Report** - Bonus feature with counts per priority and status

### Additional Features
- Modern, responsive UI with animations
- Filter tasks by status
- Real-time updates without page reload
- Toast notifications for user feedback
- Modal for daily reports
- Input validation on both frontend and backend

## Technology Stack
- **Backend**: PHP 7.4+ (Native PHP with OOP)
- **Database**: MySQL
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Server**: Apache/Nginx (XAMPP/WAMP/MAMP recommended)

## Installation Guide

### Prerequisites
